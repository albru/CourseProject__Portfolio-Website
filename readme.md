Сайт Портфолио
=================

    Адаптивный сайт, состоящий из четырёх
    страниц и админки. Выполняется на третьей части курса 'Веб разработчик'. 
    Еще в разработке.


Что я использовал в этом проекте
---

* PUG
* SCSS
* BEM
* JS / jQuery
* Mobile first
* Flexbox / CSS Grid
* NPM
* Yarn
* Gulp
* Webpack
